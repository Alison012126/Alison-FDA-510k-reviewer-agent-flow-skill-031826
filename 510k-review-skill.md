# 目的與範圍
此技能旨在協助FDA官員進行MRI醫療設備的510(k)提交審查。它提供了詳細的審查指導，並生成模擬審查報告，以促進MRI設備的及時審查和市場批准。

# 何時使用 / 何時不使用
**何時使用：**
- 當您需要生成MRI醫療設備的510(k)提交審查指導時。
- 當您需要模擬審查報告以供內部參考或培訓時。

**何時不使用：**
- 不應用於非MRI醫療設備的審查。
- 不應用於需要法律意見或專業醫療建議的情況。

# 步驟工作流程
1. 確認您需要的MRI設備類型及其510(k)提交的具體要求。
2. 使用此技能生成全面的審查指導。
3. 根據生成的指導，進行詳細的審查。
4. 使用此技能生成3份模擬審查報告。
5. 審查並調整模擬報告以符合實際需求。

# 範例
## 審查指導範例
```
## MRI設備510(k)提交審查指導

### 1. 產品描述
- 產品名稱：XYZ MRI掃描儀
- 型號：XYZ-123
- 製造商：XYZ醫療科技有限公司

### 2. 510(k)提交要求
- 提交文件清單：產品說明書、風險評估報告、臨床數據
- 相關標準遵循情況：符合IEC 60601-2-33標準

### 3. 風險評估
- 主要風險：磁場影響
- 風險控制措施：設置安全區域
```

## 模擬審查報告範例
```
## 模擬審查報告

### 1. 審查概述
- 審查日期：2023年10月1日
- 審查人員：張三

### 2. 產品信息
- 產品名稱：XYZ MRI掃描儀
- 型號：XYZ-123

### 3. 審查結果
- 符合性評估：符合所有要求
- 建議：批准510(k)提交
```

# 安全性
請注意，所有生成的文檔均為模擬，並不構成法律或專業意見。請在使用前仔細檢查所有信息的準確性，並對不確定的內容進行進一步查證。避免在不受信任的環境中使用此技能，以防止信息洩露或誤用。
